### Ejercicio 2

Ingresar dos números y ofrecer al usuario un menú con las siguientes opciones:

“1-suma 2-resta 3-producto 4-división”.

Luego, mostrar el resultado de la operación aritmética elegida.

#### Pseudocódigo

    DEFINIR n1, n2,opcion
    LEER n1
    LEER n2
    LEER opcion
    SI opcion= ’ + ‘ ENTONCES
    INFORMAR n1 + n2
    FIN SI
    SINO
    SI opcion = ’ - ‘ ENTONCES
    INFORMAR n1 - n2
    FIN SI
    SINO
    SI opcion = ’ * ‘ ENTONCES
    INFORMAR n1 * n2
    FIN SI
    SINO
    SI opcion = ’ / ‘ ENTONCES
    INFORMAR n1/ n2
    FIN SI
