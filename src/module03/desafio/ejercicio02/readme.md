### Ejercicio 2

A partir del siguiente código:

int a = 10, b=-2, c=5;\
hay 2 números positivos y 1 negativo

Informar la multiplicación de los dos
números positivos.

#### Pseudocódigo

    PROGRAMA 2
    LEER a,b,c
    SI a>0 y b>0
    ENTONCES
    INFORMAR a*b
    FIN SI
    SI a>0 y c>0
    ENTONCES
    INFORMAR a*c
    FIN SI
    SI b>0 y c>0
    ENTONCES
    INFORMAR a*b
    FIN SI
    FIN PROGRAMA
